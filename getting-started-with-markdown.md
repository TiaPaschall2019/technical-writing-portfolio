# Getting Started with Markdown

## What You'll Need
- A text editor (such as Visual Studio Code, free to download)
- A file saved with a .md extension

## Step 1: Create Your File
Open your text editor and create a new file. Save it with a .md extension, such as notes.md.

## Step 2: Add a Heading
Type a single # followed by a space, then your title. This creates a large heading when previewed.

## Stepn 3: Preview Your File
Press Ctrl+Shift+V to see your formatted file rendered, instead of plain text symbols.

## Next Steps
Explore additional Markdown syntax, such as bold text, bullet lists, and links, to format more complex documents.
