# FAQ - Markdown & GitHub Basics

**Q: How do I get out of preview mode back to editing?**
A: Click back onto the editor tab, or press Ctrl+Shift+V again to togggle the preview closed.

**Q: Why isn't my Markdown link working?**
A: The link text must exactly match the file's actual name, including capitalization and file extension. Double-check for typos or extra extensions like .md.md.

**Q: Do I need to write any code before my Markdown text?**
A: No, Markdown files are plain text - you just start typing, using symbols like # and * to add formatting.

**Q: How do I know if my file saved correctly?**
A: Press Ctrl+S to save, then Ctrl+Shift+V to preview. If the preview shows formatted text instead of raw symbols, it saved correctly.
