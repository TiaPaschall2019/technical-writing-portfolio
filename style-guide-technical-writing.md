# Style Guide - Technical Writing Basics

## Voice & Tone
- Write in a clear, neutral tone. Avoid humor or overly casual language.
- Address the reader directly using "you" (e.g., "Click the button" rather than "The user should click the button").

## Formatting Conventions
- Start each step with a numbered list with an action verb (e.g., "Click," "Save," "Open").
- Use bullet points for lists that don't require a specific order.
- Keep paragraphs short - a few sentences at most.

## Word Choice 
- Avoid vaque quantifiers like "some" or "a lot." Use specific amounts or details instead. 
- Use consisitent terminology throughout a document - don't switch between different words for the same thing.
