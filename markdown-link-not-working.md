# Markdown Link Not Working

## Problem
Clicking a Markdown link (e.g., in a README) leads to an error page instead of opening the linked file.

## Cause
Markdown links are case-sensitive and must exactly match the file's actual name. A mismatch, such as an extra file extension or incorrect capitalization, will break the link.

## Soltion 
1. Go to your respitory and click on the linked file.
2. Click the pencil icon to edit the file.
3. Correct the file name in the edit box so it exactly matches the filename used in the link.
4. Add a commit message describing the fix. 
5. Click "Commit changes" to save.
